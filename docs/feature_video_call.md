# 機能仕様書：ビデオ通話相談 & 待機中タスク（タイミー風）

## 1. 概要
「センパイ・リンク」における、大学生チューターと受験生のビデオ通話マッチング機能。および、チューターの待機時間を有効活用するマイクロタスク機能の定義。

## 2. ビデオ通話の料金・時間ロジック
- **基本単位**: 1セッション 20分。
- **基本料金**: 2,000円（仮設定）。
- **延長ロジック**: 
  - 20分終了の3分前に生徒側へ通知。
  - 生徒の承認により10分単位で自動延長。
- **決済**: Stripe Connectを使用。通話終了時に最終金額を確定し、決済を実行。

## 3. マッチング・ステータス管理
- **今すぐ相談（オンデマンド）**:
  - チューター側の「待機中（オンライン）」スイッチをトリガーに、生徒側画面へリアルタイム表示。
  - Supabase Realtimeを使用して `tutor_status` テーブルを監視。
- **指名予約**: 
  - カレンダー連携による事前予約枠の設定。

## 4. 待機中の「隙間バイト（タイミー風）」
- **コンセプト**: ビデオ通話の待機時間を「時給が発生する作業時間」に変える。
- **タスクメニュー**:
  1. 新着体験記の校閲・承認
  2. 掲示板（Q&A）への回答
  3. 広報用コンテンツの作成（体験記の要約など）
- **割り込み制御**: ビデオ通話のリクエスト（着信）が来た場合、実行中のタスクは自動保存され、強制的に通話画面へ遷移する。

## 5. UI/UX 改善指針
- **脱・チープ化**: 
  - Tailwind CSS を活用し、カードにソフトなドロップシャドウ（shadow-lg）と薄いボーダーを適用。
  - ボタンには適切なトランジションとホバーエフェクトを追加し、信頼感のある「知的なデザイン」を目指す。
